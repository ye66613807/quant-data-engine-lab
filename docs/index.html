第 1 章：数据引擎

Source

从 CSV 开始接入行情数据，后续可以替换为数据库、交易所 API 或第三方数据服务。

Model

统一 OHLCV 数据结构，让后续研究代码不被数据来源绑住。

Engine

提供 symbols、history、latest 等查询接口，为回测和因子分析准备稳定入口。

Cache

预留本地缓存层，减少重复读取和未来的网络请求成本。
